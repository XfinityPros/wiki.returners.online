# Development Roadmap

This roadmap shows the planned development direction for **Returners Online**.

## Status Key

| Status         | Meaning                   |
| -------------- | ------------------------- |
| ✅ Done        | Completed                 |
| 🟡 In Progress | Currently being worked on |
| 🔵 Planned     | Scheduled but not started |
| ⚪ Later       | Planned in to do list     |
| 🔴 Delayed     | Blocked or postponed      |

## Roadmap

| Feature              | Status | Notes                                                |
| -------------------- | -----: | ---------------------------------------------------- |
| Game concept         |     🟡 | Core idea and direction defined                      |
| Game wiki            |     🟡 | Basic documentation site in progress                 |
| Royal City           |     🔵 | Main safe city where players start                   |
| Character creation   |     🔵 | Basic player setup and identity                      |
| Equipment system     |     🔵 | Bag, cape, relic, ring and mount                     |
| Stats system         |     🔵 | Health, stamina, wisdom, mana, favor and corruption  |
| Inventory system     |     🔵 | Items, resources, equipment, food, and potions       |
| First fantasy world  |     🔵 | First playable summoned world outside the Royal City |
| Combat system        |     🔵 | Basic PvE fighting system                            |
| Currency system      |     🔵 | Diamond, Gold, and Shards                            |
| Player economy       |     ⚪ | Trading, marketplace/black market, and item pricing  |
| Diamond exchange     |     ⚪ | Buy and sell Diamond using in-game currencies        |
| Premium subscription |     ⚪ | Premium perks without making the game unfair         |
| Marriage system      |     ⚪ | Social system connected to rings and player identity |
| PvP zones            |     ⚪ | Battle Arenas and PvP enabled resource sanctuaries   |
| 1v1 Duels            |     ⚪ | Player duels within the royal city                   |
| Title mastery        |     ⚪ | Long-term character progression                      |
| Cosmetics            |     ⚪ | Unlockable and buyable visual items                  |
| More worlds          |     ⚪ | New worlds and gods added over time                  |
| Companion web app    |     ⚪ | Leaderboards, chat, profiles, and market tools       |
| Mobile companion app |     ⚪ | Possible future app for chat and game tools          |

## Notice

This roadmap may change as development continues. Some features may be renamed, delayed, removed, or released earlier depending on testing and feedback.
